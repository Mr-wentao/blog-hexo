## dig

### 安装

```
apt-get install dnsutils

yum install bind-utils
```

### 查询

```bash
dig auth.vipthink.cn @114.114.114.114
```

## nslookup

### 检查域名A记录

```bat
nslookup baidu.com 114.114.114.114
nslookup -type=a qq.com 114.114.114.114
nslookup baidu.com
```

### 检查域名NS记录

```bat
nslookup
> set type=ns
> zahui.fan 114.114.114.114
```

### 检查域名CNAME记录

```bat
nslookup
> set type=cname
> zahui.fan 114.114.114.114
```
